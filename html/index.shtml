The National Human Genome Research Institute (NHGRI) model organism ENCODE (modENCODE) Project will try to identify all of the sequence-based functional elements in the Caenorhabditis elegans and Drosophila melanogaster genomes. The project is run as a Research Network.

See NHGRI news release, and background information about the project can be found in the two RFAs. RFA-HG-06-006 solicited applications for a set of projects that will conduct experiments to identify functional elements in the target genomes. RFA-HG-06-007 solicits applications to participate in the Research Network as a Data Coordination Center.

This site is maintained by the modENCODE Data Coordination Center

More information will appear on this site over the coming months. Please check back for descriptions of the projects, experimental protocols, and released data sets.
